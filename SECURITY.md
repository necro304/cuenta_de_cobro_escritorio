# Política de Seguridad

## Versiones Soportadas

Actualmente solo se proporciona soporte de seguridad para la versión más reciente del proyecto.

| Versión | Soportada |
| ------- | --------- |
| > 1.0.x | ✅ |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor **NO** abras un issue público. En su lugar, reporta el problema de la siguiente manera:

1.  Envía un correo electrónico a [necro304@users.noreply.github.com](mailto:necro304@users.noreply.github.com) (o utiliza la función de "Report a vulnerability" en GitHub si está habilitada).
2.  Incluye detalles sobre el error y pasos para reproducirlo.

Haremos lo posible por responder en un plazo de 48-72 horas y coordinar una solución antes de hacerla pública.
