# Guía de Contribución

¡Gracias por tu interés en contribuir a Cuenta de Cobro Electron! Como proyecto de código abierto, valoramos todas las contribuciones, desde la corrección de errores hasta nuevas características y mejoras en la documentación.

## ¿Cómo puedo contribuir?

### Reportar Errores (Bugs)

Si encuentras un error, por favor abre un [issue](https://github.com/necro304/cuenta_de_cobro_escritorio/issues) incluyendo:
- Una descripción clara del error.
- Pasos para reproducirlo.
- Comportamiento esperado vs. comportamiento actual.
- Capturas de pantalla si es posible.

### Sugerir Mejoras

Las sugerencias para nuevas características son bienvenidas. Abre un issue y describe la funcionalidad que te gustaría ver.

### Enviar Pull Requests

1.  Asegúrate de que el código sigue los estándares del proyecto (linting y formateo).
2.  Crea una rama descriptiva para tus cambios.
3.  Escribe mensajes de commit claros.
4.  Actualiza la documentación si es necesario.

## Configuración del Entorno

1.  Clona el repositorio.
2.  Instala las dependencias: `npm install`.
3.  Asegúrate de que el proyecto compila correctamente antes de empezar a trabajar.

## Estilo de Código

- Usamos **Prettier** para el formateo de código.
- Usamos **ESLint** para asegurar la calidad del código.
- Por favor, ejecuta `npm run format` y `npm run lint` antes de enviar tu PR.

¡Esperamos tus contribuciones!
